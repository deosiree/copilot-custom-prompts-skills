# 目标
围绕“4. 预测与迁移（适合长期记忆：学以致用）”这一类任务，稳定执行原 prompt 想表达的核心能力，并把零散命令沉淀为可复用的 skill。

## 适用场景
- 当需要把当前笔记迁移到实际场景，生成应用题、反直觉提醒和闪卡问题以强化长期记忆时使用。
- 适合需要复用同一类提示词、同时希望 agent 按固定结构稳定输出结果的场景。

## 输入要求
- `activeNote` 或当前笔记内容

## 输出要求
- 输出应完整满足原 prompt 中声明的格式与结构约束

## 执行步骤
1. 先识别输入材料、上下文来源和隐含占位符，明确本次处理对象。
2. 提取原 prompt 的核心任务、结构约束和禁止项，再将它们转写为稳定的执行顺序。
3. 在最终输出前逐项校验格式、语气、长度和保留项，避免漏掉原 prompt 的关键约束。

## 保留约束
以下原始约束在执行时必须继续遵守：

```markdown
你是一位实战专家。根据 {activeNote} 的内容：
1.场景模拟： 请设计一个我明天可能遇到的实际问题，必须用到本文知识才能解决。
2.反直觉点： 指出文中哪一个观点最容易被大多数人误解或忽略？
3.记忆提取： 请生成 3 个关于本文的 Flashcard（闪卡）问题，要求答案必须是文中的核心细节。
```

## 注意事项
- 将 `{}`、`{activeNote}` 一类占位符视为运行时输入，不要把占位符原样遗留在最终结果里。
