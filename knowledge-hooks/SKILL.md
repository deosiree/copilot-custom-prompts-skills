# 目标
围绕“1. 结构化解构（适合初读：建立知识挂钩）”这一类任务，稳定执行原 prompt 想表达的核心能力，并把零散命令沉淀为可复用的 skill。

## 适用场景
- 当需要围绕当前笔记做三层解构、概念关联和跨领域类比，帮助初次理解并建立知识挂钩时使用。
- 适合需要复用同一类提示词、同时希望 agent 按固定结构稳定输出结果的场景。

## 输入要求
- `activeNote` 或当前笔记内容

## 输出要求
- 输出应完整满足原 prompt 中声明的格式与结构约束

## 执行步骤
1. 先识别输入材料、上下文来源和隐含占位符，明确本次处理对象。
2. 提取原 prompt 的核心任务、结构约束和禁止项，再将它们转写为稳定的执行顺序。
3. 在最终输出前逐项校验格式、语气、长度和保留项，避免漏掉原 prompt 的关键约束。

## 保留约束
以下原始约束在执行时必须继续遵守：

```markdown
你现在是知识图谱专家。请分析笔记 {activeNote}，执行以下操作：
1.核心逻辑： 用三个层级（是什么、为什么、怎么做）提炼本文。
2.概念关联： 找出文中最重要的 3 个术语，并分别用一句话解释它们之间的因果或逻辑联系。
3.知识挂钩： 这是一个关于 [填写你熟悉的领域] 的人，他该如何将本文知识与已知知识联系起来？请给出一个类比。
```

## 注意事项
- 将 `{}`、`{activeNote}` 一类占位符视为运行时输入，不要把占位符原样遗留在最终结果里。
